# GOERS TEST

## How To Run
1. The language used for this test is Javascript (Node.js)
2. Make sure you have NodeJS installed on your local environment
3. `cd {problem_folder}`
4. `node main`
5. Below is the project structure

```
----- goers-test
|
|---- [Problem 18] - [Maximum path sum I]
|		|- main.js
|
|

```
